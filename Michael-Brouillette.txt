    1. What is Semantic HTML? 
    Semantic markup uses the markup language to convey information about the meaning of elements in a document.
    
    2. What is HTML used for? 
    It is used as the standard markup language for documents designed to be displayed in a web browser.
    
    3. What is an attribute and where do we put it?
     Attributes are special flags used when you have information that you do not want to render on a page.
     Attributes are placed within the opening tag of an element.

    4. What is the h1 tag used for? How many times should I use it on a page?
    The h1 tag is used as an important header and will be the biggest text. It should only be used once per 
    page.

    5. Name two tags that have required attributes
    a (anchor) and <img>

    6. What do we put in the head of our HTML document? 
    The style and link elements

    7. What is an id? 
    Titles that can only appear on a single element.

    8. What elements can I add an id to? 
    Any element

    9. How many times can I use the same id on a page? 
    One

    10. What is a class? 
    Class can apply to multiple elements vs the single element with id

    11. What elements can I add a class to? 
    Any element

    12. How many times can I use the same class on a page? 
    As many times as necessary

    13. How do I get my link to open in a new tab?
    target='_blank' should be inserted after the link address

    14. What is the alt attribute in the image tag used for? 
    It is used as an alternate text for an area in case the image cannot be displayed.

    15. How do I reference an id?
    #

    16. What is the difference between a section and a div
    A section defines sections in a document like headers or footers. Div defines a division in the document.

    17. What is CSS used for? 
    It is used to define styles in a page like the design and layout, fonts and color. It also is used to adapt to different devices and screen types.

    18. How to we select an element? Example - every h2 on the page
    <div class = "h2"> </div>

    19. What is the difference between a class and an id? - Give me an example of when I might use each one
    Class can apply to multiple elements, while id refers to a single element. You may need to use class to change the font of multiple paragraphs. You may use id to change a single element in the header or footer.
